# 团团看盘 (TuanTuan Stocks)

A cute US-stock **price viewer** — look only, never trade. 团团 (the green bean from
[nudge](https://github.com/orzzzl/nudge)) flies a tiny plane along your stock's price line:
climbing happily on green days, diving in adorable panic on red ones, and blowing bubbles
when the price sinks below the waterline (the centered 0% baseline).
